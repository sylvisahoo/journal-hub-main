# 📓 Journal Hub

A secure, modern, and mobile-first personal journaling application built with **Flutter** and a scalable backend. Journal Hub helps users capture daily thoughts, organize memories, track writing habits, and reflect on personal growth while ensuring privacy and security.

---

# 🎯 Project Goal

Deliver a secure, scalable, and intuitive digital journaling experience that empowers users to privately record, organize, search, and reflect on their daily lives while maintaining high standards of security, usability, and performance.

---

# 🛠️ Technology Stack

| Layer    | Technology                        |
| -------- | --------------------------------- |
| Frontend | Flutter (Mobile + Web)            |
| Backend  | Node.js v24.16.0 + Express v5.2.1 |
| Database | SQLite v3.52.0                    |

# 📋 Development Workflow

The project follows a **module-by-module vertical slice development** approach.

1. Design UI (Stitch)
2. Implement Flutter UI
3. Validate using Mock Data
4. Implement Backend Module
5. Integrate APIs
6. Execute QA Testing
7. Fix QA Findings
8. Review Module
9. Repeat for Next Module
